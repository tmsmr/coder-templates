# coder-templates

## Quickstart
- [Install Coder](https://coder.com/docs/coder-oss/latest/install) on your client
- `coder login https://coder.example.com`
- `cd $template`
- `coder template create`

More information: https://coder.com/docs/coder-oss/latest/templates
